# Scientific Calculator GUI

A desktop Scientific Calculator application built using Java Swing. 

## Features
- **Basic Operations:** Addition, Subtraction, Multiplication, Division.
- **Scientific Functions:** Trigonometric functions (Sin, Cos, Tan in Degrees), Square Root, Logarithm (log10), Natural Logarithm (ln), and Power ($x^y$).
- **User Interface:** Clean GUI layout with categorized action buttons.

## How to Run
1. Download the `ScientificCalculator.java` file.
2. Open CMD in the file directory.
3. Compile using: `javac ScientificCalculator.java`
4. Run using: `java ScientificCalculator`
